## ⚙️ **Quick Setup for Daily Reports**

Add this to your Homebridge config:

```json
{
  "platforms": [
    {
      "platform": "AhoyDTU",
      "name": "AHOY-DTU Solar",
      "mqttHost": "192.168.1.100",
      "usePreset": "detailed",
      "dailyReports": {
        "enabled": true,
        "language": "de",
        "reportStyle": "motion",
        "reportTime": "sunset+30"
      }
    }
  ]
}
```

**Language Options:**
- `"en"` - 🇺🇸 English
- `"de"` - 🇩🇪 Deutsch  
- `"fr"` - 🇫🇷 Français
- `"it"` - 🇮🇹 Italiano
- `"zh"` - 🇨🇳 中文
